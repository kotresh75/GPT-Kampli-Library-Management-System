## How to Insert Diagrams in Word using Draw.io Add-in

### **Step 1: Install Draw.io Add-in (if not already installed)**
1. Open Microsoft Word
2. Go to **Insert** → **Get Add-ins**
3. Search for **"draw.io"** or **"diagrams.net"**
4. Click **Add** to install

### **Step 2: Insert Diagram in Word**
1. In Word, go to **Insert** → **My Add-ins**
2. Select **draw.io**
3. Click **Open Existing Diagram**
4. Browse to the `.drawio` file
5. The diagram will be inserted as an editable image

### **Step 3: Edit Diagram (Optional)**
1. Click on the inserted diagram
2. The draw.io editor will open
3. Make any changes needed
4. Click **Save & Exit**

### **Step 4: Export as Image (Alternative)**
1. Open the `.drawio` file in https://app.diagrams.net/
2. Go to **File** → **Export as** → **PNG**
3. Choose resolution (recommended: 300 DPI for print)
4. Save the PNG file
5. Insert PNG in Word: **Insert** → **Pictures**

---

## Recommended Approach

**For DOCX submission:**
- Use the draw.io add-in to insert diagrams directly
- Diagrams remain editable if changes are needed
- Better quality than screenshots

**For PDF submission:**
- Export each diagram as high-resolution PNG (300 DPI)
- Insert PNG images in Word
- Convert Word to PDF

---

## Alternative: Convert Mermaid to PNG

If you prefer PNG images instead of draw.io:

1. Go to https://mermaid.live/
2. Open each `.mmd` file
3. Copy the Mermaid code
4. Paste in Mermaid Live Editor
5. Click **Download PNG**
6. Insert PNG in Word

## Notes

- `.drawio` files work best with the draw.io Word add-in
- PNG files have universal compatibility
